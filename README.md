# Projet-copilot
Ce projet s'inscrit dans le cadre de la Majeure 2TD de l'ESME Ivry-sur-Seine. Il vise à développer un prototype fonctionnel de système de perception coopérative entre véhicules connectés, en combinant des techniques de vision par ordinateur (YOLOv8), de communication réseau (sockets UDP/TCP) et de simulation réseau (Mininet-WiFi).
